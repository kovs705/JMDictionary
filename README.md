# JMDictionary App

Pet-project app using [**JMDict-Simplified**](https://github.com/scriptin/jmdict-simplified) Search, Save and Learn Japanese words!

## Stack
 - MVVM + MVI
 - [The JMDict Project](https://www.edrdg.org/jmdict/j_jmdict.html)
 - SwiftUI
